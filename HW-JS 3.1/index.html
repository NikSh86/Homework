Задание 2

1. Функция сложения 2-х объектов-дробей.

2. Функция вычитания 2-х объектов-дробей.

3. Функция умножения 2-х объектов-дробей.

4. Функция деления 2-х объектов-дробей.

5. Функция сокращения объекта-дроби.
